# <a name="known-issues-surface-hub"></a>Известные проблемы: Surface Hub

В этой статье перечислены известные проблемы с Surface Hub, работающими под управлением текущей операционной системы Windows 10 для совместной работы обновления 2020 г.

Чтобы Surface Hub получать последние обновления, **** >  войдите с помощью учетной записи администратора и выберите "Все приложения **Параметры** >  **Обновления** >  и безопасность **клиентский компонент Центра обновления Windows** а затем установите все обновления.




| Проблема               | Описание           | Средство                 |
|---------------------|-----------------------|------------------------|
| При использовании приложения доски Surface Hub устройствах содержимое не может быть предоставлено по электронной почте.             | При выполнении процесса экспорта доски для отправки по электронной почте содержимого приложения доски на Surface Hub устройствах отображается сообщение "Ваше устройство не настроено для электронной почты".  В результате содержимое доски не может быть предоставлено по электронной почте.                                                                                                                                                                                                                   | Корпорация Майкрософт знает об этой проблеме и активно изучает ее.  Корпорация Майкрософт предоставит дополнительные сведения о разрешении как можно быстрее.                                                                                                                                                                                                                                                                                                                                                                   |
| Некоторые Surface Hub перезапускаются, когда пользователь выбирает "Завершить сеанс".                                                                      | Когда Surface Hub устройства выбирают функцию "Завершить сеанс", чтобы очистить данные пользователя, устройство Surface Hub может ошибочно обнаружить сбой очистки, принудив перезапуск Windows, чтобы убедиться, что очистка выполнена успешно.                                                                                                                                                                      | Корпорация Майкрософт знает об этой проблеме и активно изучает ее.  Корпорация Майкрософт предоставит дополнительные сведения о разрешении как можно быстрее.                                                                                                                                                        |
| При использовании Устройств Surface Hub GCC средах с высоким уровнем доступности присоединение к собранию одним щелчком из календаря экрана приветствия не работает. | Если щелкнуть "Присоединиться" в календаре Surface Hub для Teams собрания в среде GCC High, собрание не запускается автоматически. | Чтобы присоединиться к собранию, запустите Teams, а затем присоединитесь к собранию из Teams, отображаемой клиентом.<br> <br>Корпорация Майкрософт также активно изучает эту проблему и предоставит дополнительные сведения о разрешении как можно быстрее. |
| Параметры качества обслуживания (QoS) не работают должным образом | После настройки параметров качества обслуживания с помощью политики MDM или пакета подготовки метки DSCP не применяются к трафику Teams или Skype для бизнеса (SfB). | Корпорация Майкрософт знает об этой проблеме и активно изучает ее.  Корпорация Майкрософт предоставит дополнительные сведения о разрешении как можно быстрее. |
| Синхронизация календаря учетных записей гибридных устройств с локальными почтовыми ящиками завершается сбоем.   | Surface Hub устройства по умолчанию используют современную проверку подлинности для учетных записей, существующих в Azure AD, даже если у них есть почтовые ящики в локальных средах, для которых не включена гибридная современная проверка [подлинности.](/microsoft-365/enterprise/configure-exchange-server-for-hybrid-modern-authentication) В этом сценарии Exchange прекращает синхронизацию собраний с устройством. В результате устройство не получает и не отображает новые собрания.                                                                                                    | После установки [KB4598291](https://support.microsoft.com/help/4598291) (или последующего Windows CU) поставщик служб конфигурации [SurfaceHub](/windows/client-management/mdm/surfacehub-csp) имеет новый параметр ExchangeModernAuthEnabled, доступный для переключения использования современной проверки подлинности. Для этого можно задать значение false с помощью политики MDM [](https://download.microsoft.com/download/8/3/F/83FD5089-D14E-42E3-AF7C-6FC36F80D347/ExchangeModernAuthDisabled.ppkg) или пакета подготовки, чтобы запретить центру использовать современную проверку подлинности.                                                                                                |
| Небольшое подмножество устройств Surface Hub версии 1 не может автоматически обновиться до обновления Windows 10 для совместной работы 2020 года.                                            | Это небольшое подмножество устройств Surface Hub версии 1 находится в состоянии, которое препятствует совместимости с прямым обновлением через клиентский компонент Центра обновления Windows.                                                                                                                                          | Вручную измените образ устройства на Windows 10 для совместной работы обновления 2020 с помощью Surface Hub [восстановления](surface-hub-recovery-tool.md).                                                                                                                                                                                 |
| Surface Hub отображается сообщение "Нет загрузочного устройства" после попытки установить обновление Windows 10 для совместной работы 2020 г.                                                                        | В клиентский компонент Центра обновления Windows 2020 Windows 10 для совместной работы некоторые устройства Hub версии 1 будут перезагружаться.                                                                                                                                                                                                                                       | Вручную измените образ устройства на Windows 10 для совместной работы обновления 2020 с помощью Surface Hub [восстановления](surface-hub-recovery-tool.md).                                                                                                                                                          |
| Устройства Концентратора 2S не могут получать обновления драйверов с помощью служб WSUS.                                             | Surface Hub 2S поддерживает клиентский компонент Центра обновления Windows и клиентский компонент Центра обновления Windows для бизнеса для распространения драйверов; распространение через Windows Server Update Services (WSUS) не поддерживается.                                                                                                                                                                                                                                                                      | При использовании служб WSUS перейдите на клиентский компонент Центра обновления Windows для бизнеса.<br> <br>**Дополнительные сведения**: [Windows обновления для бизнеса?](/windows/deployment/update/waas-manage-updates-wufb)                                                                                                                                                                                                                                                                                                                            |
| Центр уведомлений имеет некликируемую Параметры ссылку. | Эта ссылка не должна отображаться в Windows 10 для совместной работы и может привести к путанице.   | Функциональные возможности такие же, как и до обновления 2020 г. Раздел "Приложения" меню  должен использоваться для запуска Параметры приложения.    |
| Некоторые параметры простоты доступа сохраняются после завершения сеанса.| Когда пользователи включают переключатель "Высокая контрастность" в меню "Быстрые действия" или в приложении Параметры, этот переключатель сохраняется после завершения сеанса пользователя. Аналогичным образом, если пользователи изменяют отображение уведомлений, указав 7 секунд по сравнению с заданными администратором 5 секундами, он остается 7 секунд, даже если другие параметры сбрасываются до значений, определенных администратором.|Пользователи могут отключить переключатель высокой контрастности в меню быстрых действий(курсора), доступном на панели задач, вскоре после запуска сеанса в центре. Следующий пользователь может задать другой срок отображения уведомлений через приложение Параметры , так как этот параметр доступен для всех пользователей. Корпорация Майкрософт активно ищет решение этой проблемы.|                                  |
