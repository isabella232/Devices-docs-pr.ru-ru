# <a name="configure-windows-10-pro-or-enterprise-on-surface-hub-2"></a>Настройка Windows 10 Pro или Корпоративная на Surface Hub 2

После завершения процесса установки миграции в Windows 10 Pro или Enterprise можно выполнить следующие действия по настройке приложений и параметров на Surface Hub 2. Эти действия рекомендуется для обеспечения наилучшего опыта при использовании этого персонализированного большого сенсорного экрана и пера компьютера.

При выполнении этих действий может оказаться полезным использовать проводную или беспроводную клавиатуру и мышь.

## <a name="configure-system-settings"></a>Настройка параметров системы

1. Во входе в учетную запись с локальными привилегиями администратора на устройстве.  

    - На присоединив устройствах Azure AD пользователь, который выполняет присоединиться к Azure AD, автоматически добавляется в локализованную группу администраторов. Глобальные администраторы Azure AD и администраторы устройств Azure AD также <a href="https://docs.microsoft.com/azure/active-directory/devices/assign-local-admin" target="_blank"> являются локальными администраторами. </a> 
    
    - Вы можете ввести **чистых администраторов** локальных групп по командной подсказке, чтобы перечислить учетные записи, которые имеют права локального администратора.
    
2. Переименование устройства с использованием удобного имени, например: **username-SHub-Desktop.**

3. Выберите ****  >  **Начните Параметры**  >  **учетные**  >  **записи Синхронизируйте параметры** и отключите **параметры Синхронизации.** 

    - Параметры, используемые здесь, предназначены для обеспечения наилучшего сенсорного экрана на большом экране, поэтому вам может не потребоваться синхронизировать другие устройства.
    
4. Перезагрузите устройство.

## <a name="enable-the-touch-keyboard-and-touchpad"></a>Включить сенсорную клавиатуру и сенсорную панель

1. Выберите ****  >  **Начните Параметры**  >  **устройства печатая**и включайте показать сенсорную клавиатуру, когда она не работает в режиме планшета, и клавиатура не  >  **** **подключена.**

2. Нажмите и удерживайте или нажмите правой кнопкой мыши на панели задач, а затем выберите **кнопку Показать** сенсорную клавиатуру и **кнопку Show touchpad**. 

    - Сенсорная клавиатура полезна для прямого ввода пользователя, а виртуальная сенсорная панель помогает с точными выборами, наведении на экране подсказки или в качестве альтернативы нажатию и удержанию правой кнопкой мыши. 
    
    - См. следующий пример.

      ![Параметры касания.](images/touch.png)

3. Настройка сенсорной клавиатуры на QWERTY и плавающей.

    1. Выберите **значок Клавиатура** на панели задач, чтобы показать сенсорную клавиатуру.
    
    1. На сенсорной клавиатуре выберите значок клавиатуры в левом верхнем углу, чтобы открыть параметры клавиатуры.
    
    1. Выберите следующий тип клавиатуры в верхней строке, чтобы включить QWERTY, а последний вариант на втором ряду для обеспечения плавающего, что очень полезно на этом большом экране. См. следующие примеры.

       ![Параметры клавиатуры.](images/kbd.png)
 
4. Настройка параметров мягкой клавиатуры.

    1. Выберите **значок Параметры** на сенсорной клавиатуре или поиск и откройте **параметры ввода.**
    
       ![мягкие параметры клавиатуры.](images/sh2-softkeyboard.png)

    1. Введите все параметры в клавиатуре Spelling, Typing и Touch.


В следующем примере показан трекпад, который полезен для навигации и выбора параметров. Экранная клавиатура используется для поиска Microsoft Store:

## <a name="onedrive-for-business"></a>OneDrive для бизнеса

Используйте <a href="https://docs.microsoft.com/onedrive/onedrive" target="_blank"> OneDrive для бизнеса для легкого обмена инструментами, журналами и другими файлами </a> между всеми вашими устройствами.

- OneDrive позволяет обмениваться рабочими файлами между ноутбуками, Surface Hub desktop и мобильными устройствами с управлением Intune. Файлы можно редактировать на любом устройстве, и все подключенные к сети устройства будут обновлены с помощью изменений.

- Учитывая размер Surface Hub SSD (128 ГБ), если вы настраиваете OneDrive на устройстве Surface Hub Desktop, убедитесь, что конфигурация по умолчанию является сохранение файлов в Интернете и загрузка файлов при их использовании.

Чтобы настроить OneDrive для загрузки файлов только при необходимости, установите параметр **Files On-Demand** для сохранения пространства и скачивания файлов при **их использовании.** Дополнительные сведения см. в <a href="https://docs.microsoft.com/onedrive/files-on-demand-windows" target="_blank"> Windows. </a>

![OneDrive параметров.](images/onedrive.png)

> [!NOTE]
> Вы также можете повторить эти действия, чтобы настроить личный OneDrive, но не забудьте сохранить пространство диска и скачивать файлы только по мере их необходимости.

## <a name="sharepoint-and-teams"></a>SharePoint и Teams

SharePoint и Teams могут также локально синхронизироваться с настольными устройствами, такими как ноутбуки и концентраторы Surface, с помощью приложение синхронизации OneDrive двигателя.

Синхронизация внутренних корпоративных файлов с локальным диском с приложение синхронизации OneDrive приложением:

1. Перейдите на SharePoint и перейдите в каталог документов верхнего уровня для файлов, которые вам интересны в просмотре или редактировании с локального устройства.

2. Выберите **кнопку Sync** в верхней части SharePoint ленты.

3. Выберите на **open** в **всплывающее всплывающее место Этот**сайт пытается открыть Microsoft OneDrive .

4. Убедитесь, SharePoint файлы синхронизируются с локальным диском, выбрав на OneDrive значок в правом нижнем справа от панели задач.

5. Убедитесь, что конфигурация установлена для сохраняемой онлайн-загрузки файлов только при их использовании:

    1. Откройте проводник файлов.
    
    2. Перейдите вправо и вправо нажмите SharePoint имя; например, **Contoso \<SharePoint Document Folder Name\> \ **.
    
    3. Выберите **свободное пространство.**
    
    4. Столбец Status отображает состояние файлов и папок. Дополнительные сведения см. в <a href="https://support.microsoft.com/office/sync-sharepoint-files-with-the-onedrive-sync-client-groove-exe-59b1de2b-519e-4d3a-8f45-51647cf291cd" target="_blank"> SharePoint sync приложение синхронизации OneDrive </a> клиентом.
    
6. Teams Файлы каналов хранятся на SharePoint сайтах с одинаковыми функциями SharePoint документов, включая историю версий и синхронизацию с локальными настольными устройствами. Синхронизация Teams каналов:

    1. Перейдите к Teams интересуемого канала и выберите вкладку **Files** в верхней части. Затем выберите **Синхронизация**. Файлы начнут синхронизироваться и будут видны в Проводнике файлов на рабочем столе **\ Contoso \<name of the Teams Channel\> \ **.
    
    2. Используйте ту же процедуру, что и для синхронизации SharePoint сайтов, чтобы хранить файлы в облаке и загружать их только при их использовании, нажав и удерживая или щелкнув правой кнопкой мыши в проводнике файлов на Teams имя канала, а затем выбрав свободное место **.**

## <a name="surface-hub-pen-settings"></a>Surface Hub параметров пера

**Соедините Bluetooth Surface Hub перо**

Соедините перо, чтобы сохранить прошивку пера в курсе, установите ярлыки пера и получите сведения о заряде батареи на странице параметров Bluetooth устройства или в приложении Surface:

1. Выберите **Начните**  >  **Параметры**  >  **устройства**.

2. Выберите **добавить Bluetooth или другое устройство.**

3. Выберите **Bluetooth**.

4. Снимите кнопку пера и встряхните, чтобы отключить подключение к батарее.

5. Положите крышку обратно и нажмите и удерживайте крышку до тех пор, пока спарив светодиодные вспышки.

Вы можете установить камеру на верхней или с обеих сторон устройства. ###  <a name=&quot;fancy-zones&quot;></a>Необычные зоны


**Fancy Zones** — это часть коллекции инструментов PowerToys <a href=&quot;https://github.com/microsoft/PowerToys/releases&quot; target=&quot;_blank&quot;> на </a> GitHub.. Это отличный способ использовать экранную недвижимость на Surface Hub 2, давая вам возможность определять фиксированные макеты на дисплее (&quot;зоны"), а затем выбрать, какое приложение будет работать в каждой зоне. 


В [PowerToys вики](https://github.com/microsoft/PowerToys/wiki) есть инструкции по использованию и настройке каждого средства, включая [FancyZones.](https://github.com/microsoft/PowerToys/wiki/FancyZones-Overview) На высоком уровне после установки PowerToys можно выбрать или создать настраиваемый макет, а затем удерживать клавишу переноса вниз и перетаскивать или использовать клавиши клавиатуры для перемещения запущенного приложения в определенные зоны. В этом Bluetooth или USB-клавиатура и мышь, а также сенсорная клавиатура и сенсорная панель на экране.

**Power toys tips**
- Чтобы получать уведомления по электронной почте PowerToys обновления GitHub на странице, нажмите кнопку "вход" в верхней части [страницы](https://github.com/microsoft/PowerToys/releases).
- После PowerToys вы можете получать уведомления Windows и/или загружать и устанавливать последние обновления, настраивая параметры **** PowerToys Скачать обновления автоматически.
- Чтобы добраться до параметров PowerToys, выберите **** вверх карат Запуск приложений на панели задач, а затем правой кнопкой мыши или нажмите кнопку и удерживайте значок PowerToys до тех пор, пока меню не появится. Выберите "Параметры".
- В нижней части страницы параметров PowerToys **** автоматически включите обновления загрузки.
- Когда обновление будет выпущено, Windows появится уведомление с возможностью установки обновления.

### <a name="power-management"></a>Управление питанием

Существует несколько параметров питания для получения наилучшего использования Windows 10 Pro или Enterprise на Surface Hub 2. Это включает в себя время отключения экрана и пк и их взаимодействие со встроенным обнаружением присутствия человека (Doppler), средствами защиты от сохранения экрана и паролем, а затем, если это уместно, чтобы передать параметры питания групповой политики, предназначенные для пользователей ноутбуков и настольных компьютеров.

Windows 10 Pro или Enterprise Surface Hub 2 не позволяет экрану спать при касание, мыши и клавиатуре, а также встроенное обнаружение заполняемости человека (Doppler). Обнаружение заполняемости человека включено по умолчанию, но при желании его можно отключить в UEFI, переключив параметр устройства в средство Конфигуратора Surface UEFI либо в рамках первоначальной миграции, либо путем создания и применения более позднего пакета конфигурации UEFI. 

**Управление питанием: параметры сна на экране и ПК**

1. Выберите ****  >  **начните Параметры**  >  ****  >  **power & сна**.

2. Установите ползунок режима питания на **лучшую производительность.**

3. Настройка значений экрана и сна в ваших предпочтениях, а также учет обнаружения присутствия Doppler, которое пробуждает устройство при обнаружении движения. Соответственно, в качестве наилучшей практики рекомендуется настроить отключение экрана через **2** часа, а компьютер отключить **через 4 часа.**

**Управление питанием: за исключением экрана**

1. Поиск **параметров экрана блокировки** и **открытых параметров экрана блокировки.**

2. Настройка **параметров времени сохранения экрана** и **параметров** сохранения экрана в ваших предпочтениях. Рекомендуемые значения по умолчанию:

   - За исключением экрана (Нет) или за исключением экрана по вашему выбору.
   - Время ожидания до 15 минут.
   - В резюме отобразить экран логотипа.


**Управление питанием: групповой политики**

Перед выполнением следующей процедуры обратитесь в ИТ-отдел для утверждения, чтобы исключить Surface Hub 2S из глобальной политики управления питанием. Некоторые параметры управления питанием могут отключить функцию обнаружения присутствия.

1. Поиск центра **программного обеспечения и** его открытие.

2. Выбор **параметров**.

3. **Расширь управление power и** выберите Не применяй параметры питания из моего **ИТ-отдела на этот компьютер.**

   ![Параметры программного обеспечения.](images/soft-cntr.png)

### <a name="storage-sense"></a>Контроль памяти

В Surface Hub 2 используется SSD объемом 128 ГБ для локального хранения, поэтому необходимо учитывать использование мер по экономии хранилища при нормальном использовании.  Настройка служба хранилища Sense:

1.  Поиск **параметров хранения,** которые находятся в **параметрах System.**

2.  В **Параметры**выберите **включив** смысл хранения, чтобы открыть **страницу служба хранилища** параметров.

3.  Включаем служба хранилища sense to **On**.

4.  Выберите **Настройте служба хранилища Sense** или запустите его сейчас и настройте параметры, чтобы держать файлы в сети как можно больше (из-за ограниченного пространства дисков).

Рекомендуемые параметры:

- Запуск служба хранилища Sense = Каждый день.
- Удаление временных файлов, которые не используют мои приложения = Каждые 14 дней (по крайней мере).
- Удаление файлов в папке Загрузки, если они были там более = 30 дней.
- OneDrive. Содержимое будет работать только в Интернете, если не будет открыто более 30 дней.

## <a name="complete-and-verify"></a>Завершение и проверка

1. Сканирование и установка всех Windows обновлений.

2. Обновление групповой политики.

   1. При повышенной командной подсказке введите **gpupdate /force/boot/wait:0**.
   
3. Перезагрузите устройство.

4. Проверка приложений панели задач.

   - Подключение App
   - Значок блокировки
   - Набросок на фрагменте экрана
   - Teams (если применимо)
   - Office Приложения (если применимо)
   - Surface App
   - Доска
    
5. Проверка обнаружения присутствия.

   - Обнаружение присутствия будет зеленым значком в подносе системы.
    
6. Проверка возможности проецировать на этот компьютер включена с помощью Подключение App. После настройки **Project параметров пк** запустите приложение Подключение хотя бы один раз. (Впоследствии Подключение приложение не нужно запускать, чтобы проектов Surface Hub.)

7. Проверка параметров питания и сна.

    - Сохранение экрана: 15 минут, задав (нет), Mystify или Blank; убедитесь, что выбрано поле для запроса пароля.
    - Экран: **отключить через 2 часа**.
    - PC. **Выключите через 4 часа.**
    
8. Проверка Windows Hello работает.

9. Убедитесь, что синхронизация параметров отключена.

10. Проверка запуска приложений.
