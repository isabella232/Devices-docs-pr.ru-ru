# <a name="using-the-surface-hub-hardware-diagnostic-tool-to-test-a-device-account"></a>Использование средства диагностики оборудования Surface Hub для проверки учетной записи устройства

## <a name="introduction"></a>Введение

> [!NOTE]
> Раздел "Параметры учетной записи" средства диагностики Surface Hub оборудования не собирает никакой информации. Электронная почта и пароль, вводимые в качестве ввода, используются только непосредственно в вашей среде и не собираются и не передаются никому. Сведения о входе сохраняются только до закрытия приложения или окончания текущего сеанса Surface Hub.

> [!IMPORTANT]
> * Для запуска этого приложения не требуются привилегии администратора.
> * Результаты диагностики следует обсудить с местным администратором перед открытием вызова службы в Корпорации Майкрософт.

### <a name="surface-hub-hardware-diagnostic"></a>Surface Hub Аппаратная диагностика

По умолчанию [](https://www.microsoft.com/store/apps/9nblggh51f2g) Surface Hub аппаратное диагностическое приложение не установлено в более ранних версиях Surface Hub системы. Приложение доступно бесплатно из Microsoft Store. Для установки приложения требуются привилегии администратора.

   ![Снимок экрана аппаратной диагностики.](images/01-diagnostic.png)

## <a name="about-the-surface-hub-hardware-diagnostic-tool"></a>О средстве Surface Hub аппаратной диагностики

Средство Surface Hub аппаратной диагностики — это простой в навигации инструмент, который позволяет пользователю протестировать многие из компонентов оборудования в Surface Hub устройстве. Этот инструмент также может проверить и проверить учетную запись Surface Hub устройства. В этой статье описывается, как использовать тест Параметры учетной записи в средстве Surface Hub аппаратной диагностики.

> [!NOTE]
> Учетная запись устройства для Surface Hub должна быть создана перед любым тестированием. Руководство Surface Hub администратора содержит инструкции и сценарии PowerShell, которые помогут вам создавать учетные записи локального устройства в Интернете (Office365) или гибридных устройств. Дополнительные сведения см. в разделе Создание и тестирование учетной записи устройства [(Surface Hub)](https://docs.microsoft.com/surface-hub/create-and-test-a-device-account-surface-hub) в руководстве.

### <a name="device-account-testing-process"></a>Процесс тестирования учетной записи устройства

1. Перейдите **во все приложения,** а затем найдите Surface Hub аппаратное диагностическое приложение.

    ![Снимок экрана всех приложений.](images/02-all-apps.png)

1. Когда приложение запускается, страница **Welcome** предоставляет текстовое окно для документировать причину, по которой вы тестируете концентратор. Эта заметка может быть сохранена в USB вместе с результатами диагностики по завершении тестирования. После завершения ввода заметки выберите кнопку **Продолжить.**

    >[!NOTE]
    >При сохранении результатов диагностики не измените путь по умолчанию или выберите поднаправление. Файлы можно скопировать позже с помощью приложения File Explorer.

    ![Снимок экрана Welcome.](images/03-welcome.png)

1. На следующем экране вы можете протестировать все или некоторые Surface Hub компонентов. Чтобы начать тестирование учетной записи устройства, выберите **значок Результатов** тестирования.

    ![Снимок экрана тестовых параметров.](images/04-test-results-1.png)

    ![Снимок экрана результатов тестирования.](images/05-test-results-2.png)

1. Выберите **учетную запись Параметры**.  

    ![Снимок экрана Параметры.](images/06-account-settings.png)

    Экран Параметры учетной записи используется для тестирования учетной записи устройства.

    ![Снимок экрана Параметры учетной записи.](images/07-account-settings-details.png)

1. Введите адрес электронной почты учетной записи устройства. Пароль необязательный, но рекомендуется. Выберите **кнопку Тестовая учетная** запись, когда вы будете готовы к продолжению.

    ![Снимок экрана тестовой учетной записи.](images/08-test-account.png)

1. После завершения тестирования просмотрите результаты по четырем областям тестирования. Каждый раздел можно расширить или свернуть, выбрав знак Плюс или Минус рядом с каждой темой.

    **Network**

    ![Снимок экрана Сети.](images/09-network.png)

    **Среда**

    ![Снимок экрана среды.](images/10-environment.png)

    **Сертификаты**

    ![Снимок экрана сертификатов.](images/11-certificates.png)

    **Модель доверия**

    ![Снимок экрана модели доверия.](images/12-trust-model.png)
