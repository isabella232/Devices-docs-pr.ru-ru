# Использование системы управления помещением (Surface Hub)


С устройством Microsoft Surface Hub можно использовать системы управления помещением.

Для этого необходимо подключить оборудование системы управления к Surface Hub, обычно через последовательный порт RJ11 в нижней части Surface Hub.

## Параметры терминала

Для подключения к панели системы управления помещением не нужно настраивать параметры терминала. Если вы хотите подключить ПК или ноутбук к Surface Hub и отправлять последовательные команды из Surface Hub, вы можете использовать программу эмулятора терминала, например Tera Term или PuTTY. 

| Параметр | Значение |
| --- | --- |
| Скорость (бит/с) | 115200 |
| Биты данных | No8 | 
| Стоп-биты | 1,1 |
| Четность | нет |
| Управление потоком | нет |
| Перевод строки | каждый возврат каретки |
 

## Схема подключения

Вы можете использовать стандартный разъем RJ-11 (6P6C) для подключения последовательного порта Surface Hub к системе управления помещением. Это рекомендуемый метод. Вы также можете использовать 4-жильный кабель RJ-11, но это не рекомендуется.

## Наборы команд

Системы управления помещением используют общие сценарии конференц-зала для команд. Команды передаются из системы управления по последовательному подключению на устройство Surface Hub. Команды основаны на ASCII, при этом Surface Hub уведомляет пользователя при изменении состояния.

Доступны следующие модификаторы команд. Команды заканчивают символом новой строки (/n). Ответы могут поступать в любое время после изменения состояния, не вызванного непосредственно командой управления портом.

| Модификатор | Результат |
| --- | --- |
| + | Увеличить значение |
| - | Уменьшить значение |
| = | Задать конкретное значение |
| ? | Запросы текущего значения |
 

## Питание

Surface Hub может находиться в одном из следующих состояний питания.

| Состояние | Состояние Energy Star| Описание |
| --- | --- | --- |
| до | S5 | Отключено |
| 1,1 | - | Питание включено (не определено) |
| 2 | S3 | Спящий режим |
| 5 | S0 | Готово |


В режиме замены компьютера доступны только состояния питания "Готово" и "Отключено", а меняется только экран. Порт управления невозможно использовать для включения компьютера для замены. 

| Состояние | Состояние Energy Star| Описание |
| --- | --- | --- |
| до | S5 | Отключено |
| 5 | S0 | Готово |

Для устройства управления любое состояние кроме "5 / Готово" считается отключенным. Каждая команда PowerOn приводит к двум изменениям состояния и ответам. 

| Команда | Изменение состояния| Ответ |
| --- | --- | --- |
| PowerOn | Устройство включается (дисплей и компьютер).</br></br>Служба на ПК уведомляет SMC о том, что компьютер готов. |  Power=0</br></br>Power=5 |
| PowerOff | Переход устройства в состояние "эмбиент" (компьютер включен, экран затемняется). |  Power=0 |
| Power? |  SMC сообщает последнее известное состояние питания. |  Питание=<#> |



## Яркость

Текущий уровень яркости находится в диапазоне от 0 до 100.

Изменение уровня яркости может инициироваться системой управления помещением или другой системой.

| Команда | Изменение состояния |Ответ |
| --- | --- | --- |
| Brightness+ | Контроллер управления системой (SMC) отправляет команду увеличения яркости.</br></br>Служба ПК в системе управления уведомляет SMC о новом уровне яркости. |  Brightness = 51 |
| Brightness- |  SMC отправляет команду уменьшения яркости.</br></br>Служба на ПК уведомляет SMC о новом уровне яркости. | Brightness = 50 |

##  <a name="volume"></a>Volume

Текущий уровень громкости находится в диапазоне от 0 до 100.

Изменение уровня громкости может инициироваться системой управления помещением или другой системой.

>[!NOTE]
>Команда "Volume" управляет только громкостью для режима встроенного компьютера или компьютера на замену, а не из [источников "Guest"](connect-and-display-with-surface-hub.md).

| Команда | Изменение состояния | Ответ</br>(Включено в [режиме компьютера на замену](connect-and-display-with-surface-hub.md#replacement-pc-mode)) |
| --- | --- | --- |
| Volume+ |  SMC отправляет команду увеличения громкости.</br></br>Служба на ПК уведомляет SMC о новом уровне громкости. |  Volume = 51 |
| Volume- |  SMC отправляет команду уменьшения громкости.</br></br>Служба на ПК уведомляет SMC о новом уровне громкости. |  Volume = 50 |

| Source+ |  SMC переходит к следующему активному источнику входных данных.</br></br>Служба на ПК уведомляет SMC о текущем источнике входных данных. |  Source=&lt;#&gt; |
| Source- | SMC переходит к предыдущему активному источнику входных данных.</br></br>Служба на ПК уведомляет SMC о текущем источнике входных данных. |  Source=&lt;#&gt; |
| Source? |  SMC запрашивает у службы на ПК активный источник входных данных.</br></br>Служба на ПК уведомляет SMC о текущем источнике входных данных. | Source=&lt;#&gt; |

## Ошибки

Ошибки возвращаются в формате, описанном в этой таблице.

| Ошибка | Примечания |
| --- | --- |
| Ошибка: неизвестная команда &lt;input&gt;. |  Инструкция содержит неизвестную начальную команду. Например, &quot;VOL+&quot; — это недопустимая команда, которая возвращает сообщение &quot;Ошибка: неизвестная команда VOL&quot;. |
| Ошибка: неизвестный оператор &lt;input&gt;. |  Инструкция содержит неизвестный оператор. Например, &quot;Volume!&quot; — неверная команда, которая возвращает сообщение &quot; Ошибка: неизвестный оператор '!'&quot;. |
| Ошибка: неизвестный параметр &lt;input&gt;. |  Инструкция содержит неизвестный параметр. Например, &quot;Volume=abc&quot; — неправильная команда, которая возвращает сообщение &quot;Ошибка: неизвестный параметр abc&quot;. |
| Ошибка: команда недоступна, если &lt;input&gt; выключен. |  Если устройство Surface Hub выключено, все команды, кроме команд питания, возвращают данную ошибку. Например, &quot;Volume+&quot; — неверная команда, которая возвращает сообщение &quot;Ошибка: команда недоступна, если громкость выключена&quot;. |
