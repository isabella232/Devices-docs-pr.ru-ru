# <a name="surface-diagnostic-toolkit-for-business"></a>Набор средств диагностики Surface для бизнеса

Если surface не работает должным образом, служба диагностики поверхности Microsoft набор средств (SDT) для бизнеса может помочь вам или администратору найти и решить проблемы.  SDT для бизнеса позволяет быстро исследовать, устранять неполадки и устранять проблемы с оборудованием, программным обеспечением и программным обеспечением с устройствами Surface в сети.

> [!NOTE]
> Surface Diagnostic набор средств для бизнеса создан для коммерческих устройств. Если устройство является личным устройством и не управляется вашей работой или учебной школой, вместо этого запустите набор средств surface [Diagnostic](https://support.microsoft.com/help/4037239/surface-fix-common-surface-problems-using-surface-diagnostic-toolkit).

В частности, SDT для бизнеса позволяет:

- [Настройка пакета.](#preparing-the-sdt-package-for-distribution)
- [Запустите приложение с помощью команд.](surface-diagnostic-toolkit-command-line.md)
- [Запустите несколько тестов оборудования для устранения неполадок.](surface-diagnostic-toolkit-desktop-mode.md#multiple)
- [Создание журналов для анализа проблем.](surface-diagnostic-toolkit-desktop-mode.md#logs)
- [Получение подробных отчетов, сравнивающих устройство с оптимальной конфигурацией.](surface-diagnostic-toolkit-desktop-mode.md#detailed-report)

## <a name="installing-surface-diagnostic-toolkit-for-business"></a>Установка поверхностных диагностических набор средств для бизнеса

Чтобы создать пакет SDT, который можно распространить среди пользователей в организации:

1. Во входе на устройство Surface с помощью учетной записи администратора.

2. Скачайте пакет SDT Windows установки (.msi) со страницы [Surface Tools для скачивания ИТ](https://www.microsoft.com/download/details.aspx?id=46703).

    - Для устройств Intel/AMD скачайте: **Surface_Diagnostic_Toolkit_for_Business_v2.168.139.0.msi**.
    - Для ARM устройств скачайте: **Surface_Diagnostic_Toolkit_for_Business_v2.168.139.0_x86.msi**.

3. Скопируйте .msi в предпочтительное расположение на устройстве Surface, например, появится мастер установки SDT, как показано на рисунке 1. Нажмите кнопку **Далее**.

    >[!NOTE]
    >Если мастер установки не появится, убедитесь, что вы подписаны в учетную запись администратора на компьютере.

    ![добро пожаловать в мастер установки surface Diagnostic набор средств диагностики.](images/sdt-1.png)<br/>
    *Рисунок 1. Мастер установки набор средств surface Diagnostic*

4. Когда появится мастер установки SDT, нажмите **кнопку Далее**, примите соглашение о лицензии конечных пользователей (EULA).

5. На экране Параметры установки при желании измените расположение установки по умолчанию.

6. В соответствии с типом установки выберите **Advanced**.

    >[!NOTE]
    >Стандартный параметр позволяет пользователям запускать диагностический инструмент непосредственно на устройстве Surface при условии, что они будут подписаны на свое устройство с помощью учетной записи администратора.

    ![Параметры установки. Расширенный.](images/sdt-install.png)

7. Нажмите **кнопку Далее** и нажмите **кнопку Установить**.

## <a name="installing-using-the-command-line"></a>Установка с помощью командной строки

При желании можно установить SDT по командной подсказке и установить настраиваемый флаг для установки средства в режиме администрирования. SDT содержит следующие флаги параметра установки:

- `SENDTELEMETRY` отправляет данные телеметрии в Корпорацию Майкрософт. Флаг принимается для `0` отключенных или включенных `1` . Значение по умолчанию — `1` отправка телеметрии.
- `ADMINMODE` настраивает средство, установленное в режиме администрирования. Флаг принимается для `0` режима клиента или `1` для режима ИТ-администратора. Значение по умолчанию— `0`.

### <a name="to-install-sdt-from-the-command-line"></a>Установка SDT из командной строки

1. Откройте командную подсказку и введите:

    ```console
    msiexec.exe /i <the path of installer> ADMINMODE=1. 
    ```

    **Пример.**

    ```console
    C:\Users\Administrator> msiexec.exe/I"C:\Users\Administrator\Desktop\Microsoft_Surface_Diagnostic_Toolkit_for_Business_Installer.msi" ADMINMODE=1
    ```

## <a name="preparing-the-sdt-package-for-distribution"></a>Подготовка пакета SDT для распространения

Создание настраиваемой упаковки позволяет нацелить средство на определенные известные проблемы.

1. Нажмите **кнопку > запустить**, введите **Surface** и нажмите **кнопку Surface Diagnostic набор средств для бизнеса**.

2. Когда средство откроется, нажмите **кнопку Создать настраиваемый пакет**, как показано на рисунке 3.

    ![Создание настраиваемого пакета.](images/sdt-3.png)<br/>
    *Рисунок 3. Создание настраиваемой упаковки*

### <a name="language-and-telemetry-settings"></a>Параметры языка и телеметрии

  При создании пакета можно выбрать языковые параметры или отказаться от отправки данных телеметрии в Корпорацию Майкрософт. По умолчанию SDT отправляет телеметрию в Корпорацию Майкрософт, которая используется для улучшения приложения в соответствии с [Заявлением о конфиденциальности Майкрософт](https://privacy.microsoft.com/privacystatement). Если вы хотите отклокироваться, отстань от контрольного окна при создании настраиваемой упаковки, как показано ниже. Или очистить **телеметрию отправки в microsoft** check box на странице **Параметры** установки во время установки SDT.

>[!NOTE]
>Этот параметр не влияет на минимальную телеметрию, которая автоматически хранится на серверах Майкрософт при запуске тестов и ремонтов, которые требуют подключения к Интернету, например обновления Windows обновления и обновления программного обеспечения, или предоставления обратной связи с помощью кнопок Smile или Frown в панели инструментов приложения.
