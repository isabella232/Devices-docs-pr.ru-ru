# Развертывание набора средств диагностики Surface для бизнеса

Microsoft Surface Diagnostic набор средств for Business (SDT) позволяет ИТ-администраторам быстро изучать, устранять неполадки с оборудованием, программным обеспечением и микропрограммами на устройствах Surface, а также устранять их. Вы можете выполнить ряд диагностических тестов и средств восстановления программного обеспечения в дополнение к получению анализа состояния устройства и инструкций по устранению проблем. 

В частности, SDT для бизнеса позволяет:

- [Настройте пакет.](#preparing-the-sdt-package-for-distribution)
- [Запустите приложение с помощью команд.](surface-diagnostic-toolkit-command-line.md)
- [Чтобы устранить неполадки, запустите несколько аппаратных тестов.](surface-diagnostic-toolkit-desktop-mode.md#multiple)
- [Создание журналов для анализа проблем.](surface-diagnostic-toolkit-desktop-mode.md#logs)
- [Получите подробный отчет, сравнивающий устройство с оптимальной конфигурацией.](surface-diagnostic-toolkit-desktop-mode.md#detailed-report)

## Установка surface Diagnostic набор средств для бизнеса

Чтобы создать пакет SDT, который можно распространить среди пользователей в организации:

1. Во sign in to your Surface device using the Administrator account.
2. Скачайте пакет установщика Windows SDT (MSI) со страницы загрузки [средств Surface для](https://www.microsoft.com/download/details.aspx?id=46703) ИТ-пользователей и скопируйте его в предпочтительное расположение на устройстве Surface, например на компьютере.
3. Появится мастер настройки SDT, как показано на рисунке 1. Нажмите кнопку **Далее**. 

    >[!NOTE]
    >Если мастер настройки не появляется, убедитесь, что вы вписались в учетную запись администратора на компьютере. 

    ![Добро пожаловать в мастер настройки набор средств Surface Diagnostic набор средств](images/sdt-1.png)

    *Рисунок 1. Мастер настройки набор средств Surface Diagnostic набор средств*

4. Когда появится мастер установки SDT, нажмите **кнопку "Далее"** и примите лицензионное соглашение с конечным пользователем (EULA)

5. При желании на экране "Параметры установки" измените расположение установки по умолчанию. 
6. В области "Тип установки" выберите **"Дополнительные".** 

    >[!NOTE]
    >Стандартный параметр позволяет пользователям запускать средство диагностики непосредственно на своем устройстве Surface, если они вписались на свое устройство с помощью учетной записи администратора. 
    
     ![Параметры установки: расширенные](images/sdt-install.png)

7. Нажмите **кнопку** "Далее", а затем нажмите **кнопку "Установить".** 

## Установка с помощью командной строки
При желании можно установить SDT в командной панели и установить настраиваемый флаг для установки средства в режиме администрирования. SDT содержит следующие флаги вариантов установки:

- `SENDTELEMETRY` отправляет данные телеметрии в корпорацию Майкрософт. Флаг принимается для `0` отключения или `1` включения. Значение по умолчанию — `1` отправка телеметрии.
- `ADMINMODE` настраивает средство для установки в режиме администрирования. Флаг принимается для `0` режима клиента или для `1` ИТ-администратора. Значение по умолчанию— `0`.

### Чтобы установить SDT из командной строки:

1. Откройте командную подсказку и введите:

    ```
    msiexec.exe /i <the path of installer> ADMINMODE=1. 
    ```
    **Пример.**

    ```
    C:\Users\Administrator> msiexec.exe/I"C:\Users\Administrator\Desktop\Microsoft_Surface_Diagnostic_Toolkit_for_Business_Installer.msi" ADMINMODE=1
    ```

2. Когда средство откроется, нажмите **кнопку "Создать настраиваемый пакет",** как показано на рисунке 3.

    ![Создание настраиваемого пакета](images/sdt-3.png)

### Языковые параметры и параметры телеметрии

  При создании пакета можно выбрать языковые параметры или отказаться от отправки данных телеметрии в корпорацию Майкрософт. По умолчанию SDT отправляет телеметрию корпорации Майкрософт, которая используется для улучшения приложения в соответствии с заявлением [о конфиденциальности Корпорации Майкрософт.](https://privacy.microsoft.com/privacystatement) Если вы хотите отклоировать, при создании пользовательского пакета необходимо сойтись с этого, как показано ниже. Или сняйте **параметры отправки** телеметрии в Корпорацию Майкрософт на странице **"Параметры** установки" во время установки SDT. 

>[!NOTE]
>Этот параметр не влияет на минимальную телеметрию, которая автоматически хранится на серверах Майкрософт при запуске тестов и восстановлений, для чего требуется подключение к Интернету, например обновление Windows и восстановление программного обеспечения, а также предоставление обратной связи с помощью кнопок "Смайлик" или "Нажми и работай" на панели инструментов приложения.
