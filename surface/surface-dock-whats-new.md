# <a name="whats-new-in-surface-dock-2"></a>Новые возможности в Surface Dock 2

Surface Dock 2— док-станция Surface следующего поколения позволяет пользователям подключать внешние мониторы и несколько периферийных устройств для полностью модернизированного рабочего стола с устройства Surface. Встроенный для максимальной эффективности в офисе, в гибком рабочем пространстве или дома Surface Dock 2 имеет семь портов, включая два порта USB-C с двумя передними портами с мощностью быстрой зарядки для телефонов и аксессуаров мощностью 15 ватт.

## <a name="full-device-management-support"></a>Полная поддержка управления устройствами

Surface Dock 2 предназначен для упрощения ИТ-управления, что позволяет администраторам автоматизировать обновления прошивки с помощью Windows обновления или централизовать обновления с помощью внутренних средств распространения программного обеспечения.

- Режим Enterprise surface (SEMM) позволяет ИТ-администраторам обеспечивать безопасность портов на surface Dock 2. Дополнительные сведения см. в [пункте Порты Secure Surface Dock 2 с режимом управления Enterprise Surface.](secure-surface-dock-ports-semm.md)
- Windows управления инструментами (WMI) позволяет ИТ-администраторам удаленно отслеживать и управлять последними прошивками, состоянием политики и связанными данными на устройствах Surface Dock 2. Дополнительные сведения см. в [ссылке Управление surface Dock 2 с WMI](surface-dock2-wmi.md).
- Централизация обновлений в локальной сети с помощью средств распространения программного обеспечения. [Скачайте прошивку Surface Dock 2 и драйверы](https://www.microsoft.com/download/details.aspx?id=101317).

## <a name="streamlined-device-management"></a>Оптимизация управления устройствами

Surface выпустила упрощенные функции управления Windows update, что позволяет ИТ-администраторам использовать следующие функции корпоративного уровня:

- **Фрикционные обновления**. Обновите доки тихо и автоматически с помощью Windows или Microsoft Endpoint Configuration Manager (ранее System Center Configuration Manager - SCCM) или других средств развертывания MSI.
- **Пробуждение из сети**. Управление и доступ к корпоративным устройствам, не зависят от пользователей, чтобы сохранить их устройства на питание. Даже если пристыкованное устройство находится в режиме сна, спячки или отключения питания, ваша команда может будиться из сети для обслуживания и управления с помощью endpoint Configuration Manager или других средств управления предприятием.
- **Централизованный ИТ-контроль**. Управление, которые могут подключаться к Surface Dock 2, включив порты и отключить. Ограничить использование устройств-хостов с Surface Dock 2. Ограничить доступ док-станции к одному пользователю или настроить доки для доступа только определенным пользователям в вашей команде или во всей компании.
