# <a name="azure-virtual-desktop-on-surface"></a>Виртуальный рабочий стол Azure на Surface

## <a name="introduction"></a>Введение

Виртуальный настольный компьютер Azure на поверхности позволяет запускать инфраструктура виртуальных рабочих столов (VDI) на устройстве Surface , размывая границы между локальным рабочим столом и виртуальным рабочим столом, где сенсорный, перо, перо, и биометрическая проверка подлинности охватывают как физические, так и виртуальные среды. Представляющий собой еще одну веху в эволюции вычислений, Azure Virtual Desktop on Surface <a href="#1"><sup>1</sup></a> объединяет Microsoft 365 — виртуализированную в облаке Azure — с расширенными средствами защиты безопасности, управляемостью на корпоративном уровне и расширенными средствами производительности Windows 10 или Windows 11 на Surface. Это сочетание форм-факторов премиум-класса и инфраструктура виртуальных рабочих столов Azure предоставляет организациям расширенный пользовательский интерфейс, портативность, безопасность, непрерывность бизнеса и современное управление.

### <a name="azure-virtual-desktop"></a>Виртуальный рабочий стол Azure

Azure Virtual Desktop (AVD) — это комплексная служба виртуализации настольных компьютеров и приложений, запущенная в облаке Azure. Это единственная виртуальная инфраструктура настольных компьютеров, которая обеспечивает упрощенное управление, много сеансы Windows 10 или Windows 11, оптимизацию для Приложения Microsoft 365 и поддержку сред [](https://www.microsoft.com/microsoft-365) удаленного рабочего стола (RDS). С помощью AVD можно быстро развернуть и масштабировать Windows и приложения в Azure и получить встроенные функции безопасности и соответствия требованиям.

### <a name="azure-virtual-desktop-partner-integrations"></a>Интеграция партнеров Azure Virtual Desktop

Список утвержденных поставщиков партнеров и независимых поставщиков программного обеспечения можно найти в [Azure Marketplace](https://azuremarketplace.microsoft.com) и поиска для Azure Virtual Desktop. Некоторые партнеры также предоставляют виртуальный рабочий стол в качестве службы (DaaS). DaaS освободит вас от необходимости поддерживать собственные виртуальные машины (виртуальные машины), предоставляя полностью управляемый настольный компьютер подворотни и службу виртуализации. Возможность поставлять настраиваемые настольные компьютеры пользователям во всем мире позволяет компаниям быстро адаптироваться к изменяющихся рыночных условиях, накручивать облачные настольные компьютеры по запросу — когда и где они необходимы.

### <a name="offline-and-on-device-access-for-more-productive-experiences"></a>Автономный и доступный на устройстве доступ для более продуктивной работы

Традиционно решения VDI работают только тогда, когда конечная точка подключена к Интернету. Но что происходит, когда интернет или питание недоступны?

Для обеспечения непрерывности бизнеса и обеспечения продуктивности сотрудников корпорация Майкрософт разработала устройства Surface для расширения виртуального рабочего стола с автономным доступом к файлам, Microsoft 365 и сторонним приложениям. Традиционные приложения, такие как Microsoft Office, доступные на нескольких платформах (x86, x64, Universal Windows Platform, ARM), позволяют пользователям оставаться продуктивными в автономном режиме. Файлы из виртуальной облачной среды рабочего стола можно синхронизировать локально на Surface OneDrive для бизнеса для автономного доступа. Вы можете быть уверены в том, что вся кэшированная на локальном уровне информация устарела и защищена.

Помимо добавления поддержки автономного доступа к приложениям и файлам, устройства Surface предназначены для оптимизации совместной работы Microsoft Teams "On-Device". Хотя некоторые решения VDI поддерживают использование Teams виртуального сеанса, пользователи могут воспользоваться более оптимизированной поддержкой локально установленного экземпляра Teams. Локализация приложений связи и совместной работы для мультимедийных каналов, таких как голос, видео, живая подпись позволяет организациям в полной мере воспользоваться возможностью устройств Surface предоставлять оптимизированные Microsoft 365 опытом. Появление Surface AI или "AI-on-device" привносит в жизнь новые возможности, такие как технология глазного взгляда (доступная на Surface Pro X), которая регулирует внешний вид ваших глаз, поэтому зрители видят, как вы смотрите непосредственно в камеру при общении с помощью видео.

Альтернатива локальной установке традиционных приложений заключается в том, чтобы воспользоваться последней версией Microsoft Edge, которая поставляется с поддержкой прогрессивных веб-приложений (PWA). PWAs — это только веб-сайты, которые постепенно улучшаются, чтобы функционировать как родные приложения на поддерживающих платформах. Качества приложения PWA лучших веб-приложений и местных приложений с дополнительными функциями, такими как push-уведомления, обновление фоновых данных, поддержка автономного доступа и другие.

### <a name="virtual-gpus"></a>Виртуальные GPUs

Графические интерфейсы идеально подходят для рабочих нагрузок с высокой производительности и графикой, помогая клиентам подпитывать инновации с помощью таких сценариев, как высококачественная удаленная визуализация, глубокое обучение и прогностическая аналитика. Однако это не идеально для специалистов, которым необходимо работать удаленно или на работе, так как к физическим устройствам привязаны различные степени внутренней лошадиной силы GPU, что ограничивает мобильность и гибкость.

Чтобы решить эту проблему, Azure предлагает семейство виртуальных машин N-series с возможностями GPU NVIDIA (vGPU). С помощью vGPUs ИТ-функции могут обмениваться производительностью GPU на нескольких виртуальных машинах или нагрузками, требующим требовательных к мощности, назначив несколько GPUs одной виртуальной машине. Для Surface это означает, что независимо от того, какое устройство вы используете, от Surface Go 3 до Surface Laptop 4, ваше устройство имеет доступ к мощной производительности графики класса сервера. Интерфейсы Surface и vGPUs позволяют сочетать все то, что вам нравится в Surface, включая перо, сенсорную панель, клавиатуру, трекпад и PixelSense дисплеи, с возможностями графики, доступной только в высокопроводительных вычислительных средах.

Azure N-series привносит эти возможности в жизнь на устройстве Surface, что позволяет вам работать любым способом, где бы вы ни были. [Дополнительные новости о размерах виртуальных машин, оптимизированных для Azure N-Series и GPU.](/azure/virtual-machines/sizes-gpu)

### <a name="security-and-management"></a>Безопасность и управление

От чипа до облака Microsoft 365 и Surface помогают организациям оставаться защищенными и в курсе.
Благодаря оборудованию и программному обеспечению Surface, разработанным, созданным и протестированным корпорацией Майкрософт, пользователи могут быть уверены, что они производительны и защищены ведущими технологиями от чипа до облака. По мере удаленной работы пользователей корпоративные данные и защита интеллектуальной собственности становятся как никогда приоритетными. Виртуальный настольный компьютер Azure на поверхности разработан вокруг модели безопасности с нулевым доверием. Каждый запрос доступа строго проверяется, авторизируется в рамках ограничений политики и проверяется на аномалии перед предоставлением доступа.

Благодаря максимальной эффективности облачных вычислений современное управление позволяет ИТ-службе лучше обслуживать потребности пользователей, заинтересованных лиц и клиентов в условиях все более конкурентоспособной бизнес-среды. Например, устройства Surface можно запускать и запускать с минимальным взаимодействием с командой. Установка является автоматической и самообслуживаемой. Обновления являются быстрыми и безболезненно как для вашей команды, так и для пользователей. Вы можете управлять устройствами независимо от их физического расположения.

Функции безопасности и управления, поставляются с помощью Виртуального рабочего стола Azure на Поверхности:

- **Центр обновления Windows.** Постоянное Windows помогает вам опережать новые угрозы безопасности. Windows 10 и Windows 11 были спроектированы с нуля, чтобы быть более безопасными и использовать новейшие аппаратные возможности для повышения безопасности. Благодаря специально построенной версии UEFI <a href="#5"><sup>5</sup></a> и Windows для бизнеса, которая отвечает на развивающиеся угрозы, безопасная и упрощенная защита.
- **Шифрование оборудования.** Шифрование устройств позволяет защитить данные на поверхности, поэтому к ним могут получить доступ только уполномоченные лица. Все устройства Surface для бизнеса имеют дискретный модуль доверенных платформ (dTPM), защищенный от вторжения с аппаратной защитой, в то время как программное обеспечение использует защищенные ключи и измерения для проверки правильности программного обеспечения.
- **Microsoft Defender** [Microsoft Defender for Endpoint](/microsoft-365/security/defender-endpoint/microsoft-defender-endpoint?) объединяет машинное обучение, анализ больших данных, углубленное исследование устойчивости к угрозам и облачную инфраструктуру Майкрософт для защиты устройств. Средство встроено и не нуждается в дополнительных агентах, которые будут развернуты на устройствах или в среде VDI, что упрощает управление и оптимизацию запуска устройств.
- **Съемные диски.** Несколько новых устройств Surface имеют съемные диски SSD6<a href="#6"><sup></sup></a>, обеспечивающие более полный контроль над хранением данных.
- **Современная проверка подлинности.** Microsoft 365 Surface — это единая платформа, которая предоставляет все Windows безопасности (с учетом лицензирования и обеспечения). Все устройства портфеля Surface оснащены встроенной камерой, предназначенной для Windows Hello для бизнеса, обеспечивая биометрическую безопасность, которая обеспечивает бесперебойную поддержку от работы на устройстве до проектов на основе VDI.
- Современный интерфейс конфигурации микропрограммных **программ (** DFCI),<a href="#7"><sup> 7</sup></a> ИТ-администраторы могут удаленно отключать элементы оборудования на уровне микропрограмм, таких как микрофоны, порты USB, слоты SD-карт, камеры и Bluetooth, которые удаляют питание на периферийном устройстве. Защитник Windows Credential Guard использует безопасность на основе виртуализации, чтобы к ним могли получать доступ только привилегированные системные программы.
- **Обратная и вперед совместимость.** Windows 10 и Windows 11 устройств обеспечивают обратную и вперед совместимость между оборудованием, программным обеспечением и службами. Корпорация Майкрософт имеет сильную историю сохранения устаревшей поддержки оборудования, периферийных устройств, программного обеспечения и служб при внедрении новейших технологий. Предприятия могут планировать ИТ-инвестиции, чтобы иметь длительный срок службы.
- **Мост для устаревших Windows 7 рабочих нагрузок.** Для сценариев решений, зависящих Windows среды ОС, предприятия могут использовать экземпляры VDI Windows 7, запущенные в Azure. Это позволяет поддерживать на современных устройствах, таких как Surface, без риска полагаться на более старые Windows 7 машин, которые больше не получают последние обновления безопасности. В дополнение к этим преимуществам "проверки будущего" миграция любых устаревших рабочих нагрузок значительно упрощается при развертывании современного Windows 10 или Windows 11 аппаратных средств.
- **Zero-Touch развертывание.** Автопилот — это рекомендуемый современный вариант развертывания управления для устройств Surface. Windows Autopilot on Surface — это облачная технология развертывания в Windows 10 и Windows 11. Вы можете использовать Windows автопилот на Поверхности для удаленного развертывания и настройки устройств в процессе нулевого касания прямо из окна. Windows устройства, зарегистрированные на автопилоте, идентифицированы через Интернет при первом запуске с помощью уникальной подписи устройства, называемой аппаратным хашом. Они автоматически регистрируются и настраиваются с помощью современных решений управления, таких как Azure Active Directory (Azure AD) и управления мобильными устройствами.

### <a name="surface-devices-minimizing-environmental-impacts"></a>Устройства Surface: минимизация воздействия на окружающую среду

Surface выполняет оценки жизненного цикла для вычисления воздействия устройств на окружающую среду на ключевых этапах жизненного цикла продукта, что позволяет Корпорации Майкрософт минимизировать эти последствия. Каждый продукт Surface имеет профиль ECO с данными о выбросах парниковых газов, потреблении первичной энергии и составе материалов, упаковке, переработке и связанных с ними критериях. Чтобы скачать профили для каждого устройства Surface, см. [в странице ECO Profiles](https://www.microsoft.com/download/details.aspx?id=55974).

## <a name="summary"></a>Сводка

Azure Virtual Desktop on Surface обеспечивает организациям большую гибкость и устойчивость в деле обеспечения разнообразных потребностей пользователей, заинтересованных сторон и клиентов. Запуск решений Azure Virtual Desktop на устройствах Surface предоставляет уникальные преимущества по сравнению с сохранением зависимости от устаревших устройств. Гибкие форм-факторы, такие как Surface Go 3 и Surface Pro 8, подключенные к облаку (или в автономном режиме), позволяют пользователям работать в любом месте и в любое время. Независимо от того, работают ли сотрудники в настойчивых сценариях, по запросу или по времени, Azure Virtual Desktop on Surface предоставляет предприятиям универсальность для поддержания производительности во время сбоев в непредвиденных событиях. Используя встроенную многоуровневую безопасность и современную управляемость Windows 10 и Windows 11, компании могут воспользоваться расширяемой экосистемой облачных служб для быстрого развертывания и масштабирования Windows настольных компьютеров и приложений. Проще говоря, Azure Virtual Desktop on Surface предоставляет критически необходимые технологии организациям и предприятиям всех размеров.
